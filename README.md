## Projects

### Generative Grid — Sonik Kaleydoskop
Audiovisual generative art. A living grid of blue tones evolves in bilateral symmetry — each color change triggers a real-time synthesized note. 17-voice Web Audio synthesizer turns visuals into ambient sound.

**[Live Demo](https://illlknurbudak.github.io/creative-works/generative-grid.html)** | [Source](https://github.com/illlknurbudak/creamav)

### Pareidolia — Gordugunu Sandigin Sey
40-second interactive Three.js experience through five natural scenes (clouds, water, bark, soil, murmuration). Breath-synchronized animation with a custom pareidolia detection system.

**[Live Demo](https://illlknurbudak.github.io/creative-works/pareidolia/)** | [Source](https://github.com/illlknurbudak/080226_WC)

### Echo Masks — Algorithmic Echoes
Generative mask forms through Perlin noise flow fields and bilateral symmetry. Particles trace invisible currents, accumulating brush strokes that the brain reads as faces. Built with p5.js.

**[Live Demo](https://illlknurbudak.github.io/creative-works/echo-masks/)** | [Source](https://github.com/illlknurbudak/ECHO-MASKS)
